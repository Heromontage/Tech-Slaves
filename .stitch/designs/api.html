The Digital Architect Engine

Core Endpoints

API Status: Operational
GET /routes expand_more

List all monitored routes with health scores

GET /routes/{route_id}/health expand_more

Get detailed health score for a specific route

GET /bottlenecks expand_less

List currently active bottlenecks across the entire global network.

PARAM severity: string
PARAM limit: int
POST /optimize expand_more

Run optimization engine for rerouting suggestions

GET /streams/status expand_more

Get status of all data ingestion streams

// Request
GET https://api.sentinelflow.io/v2/routes/asia-pacific-express/health
